#Precondiciones#
-instalacion de node js, cmder, viusal code 
-instalacion de plugins personalizados para trabajo

********************* Pasos para nuevo proyecto *************************
#Paso 0: iniciacion de modules al  config    >  npm init
#Paso 1: Instalacion  node de cypress        >  npm install cypress --save-dev
#Paso 2: Instalar npx "si no esta instalado" >  npm install -g npx
#Paso 2.0: instalar plugin api               >  npm install 
#Paso 2.1: Instalar tab                      >  npm install -D cypress-plugin-tab 
#Paso 2.2: Instalar xpath                    >  npm install -D cypress-xpath
#Paso 2.3: Instalar upload                   >  npm install --save-dev cypress-file-upload
#Paso 2.4: Instalar drag and  drop           >  npm install --save-dev @4tw/cypress-drag-drop
#Paso 3: Instalar cucumber                   >  npm install @badeball/cypress-cucumber-preprocessor
#Paso 3.1: Instalar complementos de cucumber   >  npm i -D cypress @bahmutov/cypress-esbuild-preprocessor esbuild
#Paso 3.2: Instalar complementos de cucumber   >  npm install -D @esbuild-plugins/node-modules-polyfill  

#Paso 6: para la configuracion con   cucumber se anexa un  documento llamado cypressCUCUMBER.config.js 
         lo unico que se debe hacer  es   cambiar el nombre  a cypress.config.js  sin  borrarse  el original
         el cual se recomienda solo se  cambie  de nombre ya que se ejecutan de diferentes formas y puede ser
         complicado construirlo.

#Paso 7: agregar dependencia en el package.json >
                "cypress-cucumber-preprocessor": {
                    "stepDefinitions": "cypress/e2e/step_definitions/**/*.{js,ts}"
                }
                
#Paso 8: se deben reemplazar  las carpetas de  babel, badeball, bahmutov, cucumber, esbuild y  esbuild-plugin
con la finalidad de que funcione  la sentencia  and  y but en la version  12.2.0 del preprocessor y posteriores

#Paso 9: iniciar  cypress                     >  npx cypress open
